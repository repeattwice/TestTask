 serv - функция запуска сервера, и вызова функций соответсыкющих разным эндпоинтам.

endpoints - тут реализованы все функции обработки эндпоинтов и 2 вспомогательные функции, для реализации бизнес логики 

DB - тут функция подключения к бд   DBStructs - тут описаны все структуры проекта и наобходимые DTO
